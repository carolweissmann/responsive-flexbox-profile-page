# Responsive Flexbox Profile Page

A beginner-friendly exercise to practice Flexbox in CSS. This project creates a simple profile page with a header (logo and navigation) and a main section with three project cards, all aligned and responsive for mobile devices.

## Technologies Used
- HTML5
- CSS3 (Flexbox for layout)
- Responsive design with media queries

## How to Run
1. Clone the repo: `git clone https://github.com/yourusername/responsive-flexbox-profile-page.git`
2. Open `index.html` in your browser.

## Features
- Header with spaced navigation using `justify-content: space-between`
- Centered cards in the main section
- Mobile-responsive with stacked elements on small screens
